# Example: GlowLab Skincare — Micro-Influencer TikTok Campaign

## Brand Context
**Brand:** GlowLab  
**Product:** Vitamin C Brightening Serum ($48/bottle)  
**Goal:** Drive first-time purchases from Gen Z/Millennial women (22-38)  
**Budget:** $3,500 total (10 micro-influencers @ $250-$400 each + gifted product)  
**Platform:** TikTok primary, Instagram Reels secondary  
**Timeline:** 4-week campaign

---

## PROMPT 1 OUTPUT — Influencer Discovery Brief + Outreach

### Influencer Criteria Scorecard

**Tier: MICRO (10K-100K followers)**

| Criteria | Requirement |
|----------|-------------|
| Follower range | 15,000–75,000 |
| Minimum engagement rate | 4.5% (TikTok avg for micro: 5-9%) |
| Audience demographics (age) | 65%+ ages 18-35 |
| Audience geography | 60%+ US-based |
| Content pillars required | Skincare routine, clean beauty, self-care, wellness |
| Posting frequency | 3+ times/week on TikTok |
| Account age minimum | 6 months |
| Previous brand deals | Fine — not exclusive to competitor vitamin C products |

**Red Flags — Auto-Disqualify:**
- Engagement rate below 3.5% (purchased followers signal)
- Current partnership with Sunday Riley, Ole Henriksen, Drunk Elephant (direct competitors)
- Only emoji comments (bot engagement)
- No skincare content in last 30 posts
- Audience 50%+ outside US

## PROMPT 2 OUTPUT — UGC Campaign Brief

### Campaign Concept: "The Morning Glow Test"

**Concept A — "Day 14 Reveal"**
- Hook: Creator films their skin on Day 1 and Day 14 using GlowLab
- Core message: "I documented 2 weeks of using GlowLab every morning so you don't have to guess"
- Visual style: Natural morning light, no filter, genuine closeup
- CTA: "Use my code GLOW10 for 10% off — link in bio"

**Concept B — "Honest Skincare Review You Didn't Ask For"**
- Hook: "I was skeptical about another vitamin C serum so I filmed my actual reaction"
- Core message: Overcomes objection "it's just another serum" — shows texture, smell, absorption
- Visual style: bathroom counter, real lighting, unboxing moment

**Concept C — "My Dermatologist Recommended Vitamin C — This Is What I Tried"**
- Hook: Authority signal from creator's own skincare journey
- Core message: Positions GlowLab as the product that finally delivered

---

### Content Brief

**THE ONE-LINER (required in every piece):**
> "GlowLab's 15% Vitamin C serum actually makes your skin look brighter — and it doesn't sting."

**Format Specs:**
| Spec | Requirement |
|------|-------------|
| Video length | 30-90 seconds |
| Orientation | Vertical (9:16) |
| Hook (first 3 seconds) | Must create curiosity or state the problem |
| Caption length | 100-200 characters + hashtags |
| Hashtags | #ad #GlowLab #GlowLabSkin #VitaminCSerum #SkincareRoutine |

**Brand Dos:**
- ✓ Show serum texture (that golden color looks great on camera)
- ✓ Mention "15% Vitamin C" at least once
- ✓ Use in natural morning light (not studio lighting)
- ✓ Show real skin reaction — authentic is better than perfect

**Brand Don'ts:**
- ✗ Don't claim "removes dark spots" or "anti-aging" without qualifier
- ✗ Don't compare to "the $200 serum" without naming it (legal risk)
- ✗ Don't show the serum near eyes (not formulated for eye area)
