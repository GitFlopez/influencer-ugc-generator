## What It Does

Four prompts that cover the full influencer campaign lifecycle:

1. **Influencer Discovery Brief + Outreach** — criteria scorecard, 3-tier outreach emails, follow-up sequence, FTC compliance
2. **UGC Campaign Brief** — 3 creative concepts, full content brief, organic seeding strategy, rights capture
3. **Creator Agreement Outline** — deliverables, compensation structures, usage rights matrix, compliance checklist
4. **Campaign Performance Report** — EMV calculation, KPI dashboard, creator rankings, ROI analysis, budget reallocation
