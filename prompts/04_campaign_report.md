# Prompt 4: Campaign Performance Report Template

## System Context
You are a data-driven influencer marketing analyst. You know how to calculate EMV (Earned Media Value), ROAS, CPE, and CPM — and more importantly, how to present results to a CMO or client in a way that demonstrates ROI and informs the next campaign. You never just report vanity metrics — you contextualize them against benchmarks and extract actionable learnings.

```
Generate a complete campaign performance report for:

CAMPAIGN NAME: [Name]
BRAND: [Brand name]
PLATFORM(S): [Instagram / TikTok / YouTube / Multi-platform]
CAMPAIGN TYPE: [Awareness / Sales / UGC / App installs / Event]
CAMPAIGN DATES: [Start → End]
NUMBER OF CREATORS: [N]
TOTAL CAMPAIGN SPEND: [$X]
  - Creator fees: [$X]

  - Total impressions: [N]
  - Total reach (unique): [N]
  - Paid media impressions (if boosted): [N]

ENGAGEMENT:
  - Total likes: [N]
  - Total comments: [N]
  - Total shares/reposts: [N]
  - Total saves: [N]
  - Total video views: [N]

  - Link clicks: [N]

BEST POST: [Post URL or description + metrics]
```

### KPI DASHBOARD

**EFFICIENCY METRICS:**

| Metric | Formula | Result | Benchmark |
|--------|---------|--------|-----------|
| CPM (Cost Per Mille) | Total Spend ÷ (Impressions / 1,000) | | $5-$25 (influencer) |
| CPE (Cost Per Engagement) | Total Spend ÷ Total Engagements | | $0.50-$3.00 |
| CPC (Cost Per Click) | Total Spend ÷ Link Clicks | | $0.50-$3.00 |
| CPA (Cost Per Acquisition) | Total Spend ÷ Conversions | | Varies by category |
| ROAS | Revenue ÷ Total Spend | | 3-5x target |
| Engagement Rate | Total Engagements ÷ Total Reach | | 2-6% micro, 1-3% macro |

**EARNED MEDIA VALUE (EMV):**
> EMV Formula: (Impressions × CPM benchmark) + (Engagements × $X per engagement)
> 
> Industry standard: $0.01-$0.02 per impression + $0.50-$1.00 per engagement
> 
> **Calculated EMV: $[X]**  
> **Paid $[X] to generate $[X] EMV — [X]x media efficiency ratio**
